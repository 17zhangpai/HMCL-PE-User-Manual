# 安装

{% hint style="info" %}
如果安装时有问题，请到Github上向作者提交[Issues](https://github.com/Tungstend/HMCL-PE/issues)。
{% endhint %}

## 怎么安装?

就像其他应用程序的安装一样，你只需要点击安装包文件（.apk）就可以安装。

安装了HMCL-PE之后，当你第一次打开这个软件时，它会要求你启用权限，**允许**。
